# landing-page

Create high-converting landing pages with structured copywriting and opinionated HTML/CSS generation.

## Installation

```
/plugin install landing-page@devx-plugins
```

## Skills

### Writing Landing Page Copy

Generates conversion-optimized copy through a structured interview and section-by-section writing process.

**Triggers when you ask to:** write landing page copy, create marketing copy, generate web copy for a product.

**Workflow:**
1. Discovery interview to understand your product and audience
2. Section selection based on your goals
3. Copy generation for all selected sections
4. Review pass and user approval

### Building Landing Page

Generates a production-ready **Astro 5 + Tailwind CSS v4** landing page from structured copywriting content.

**Triggers when you ask to:** build a landing page, create the HTML/CSS, generate an Astro site from copy.

**Workflow:**
1. Validates copywriting input
2. Scaffolds project (Astro config, package.json, Tailwind)
3. Creates base layout and global styles
4. Generates Navbar, Footer, and section components from copy
5. Assembles index page
6. Verifies build
